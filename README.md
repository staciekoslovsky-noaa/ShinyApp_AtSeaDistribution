# Shiny App for At-sea Distribution of Marine Mammals

Project Leads: Paul Conn and Stacie Koslovsky <br>
Application Development: Christine Kwon (UChicago intern, summer 2024) <br>
Application Maintenance: Stacie Koslovsky <br>

The code and files within this repository are detailed as follows:
* **ASDShiny folder ** - code to the At-sea Distribution of Marine Mammals Shiny app
* **data folder** - contains data referenced in the Shiny app
* **inst folder** - contains code from Paul Conn to help understand and interpret the data used in the app
* **InternshipNotes_ChristineKwon.md** - documentation and information from Christine Kwon's internship
* **hfaucher_log.md** - documentation and information from Haley Faucher's internship
* **prevcode.R** - code that was developed by Christine, but ultimately not used in the application
